Technical Exercise 1

Coding a Basic HTML Page

The goal of this exercise is to refresh ourselves on writing basic HTML.


Your site should include the following:

        The following tag in the <head> section:
        <title>
        One of each of the following tags in the <body> section:
        <h1>
        <h2>
        <div>
        <p>
        <input>
        <img>
        <br>
        <ol> or <ul>
        <a>
        <button>

The above is preformatted text but these could be displayed as a list:

  • h1
  • h2
  • div
  • p
  • input
  • img
  • br
  • ol or ul (this uses the ul element)
  • a
  • button
